The tool started with java -jar gvs. It loads a gvs-store and allows executing some commands on it.

Usage: java -jar gvs [options] command

Options include:

  • --store-directory
  • --identity
  • --trust (sources>

The commands it supports:
- servicestarter (taking a graph) - uploader: updates or assert a graph (prevents lock-out if web-service missconfigured) - set-password: set the sha1 encrypted password of a user servicestarter: Take configuration graph/source, shouldn't this better be a configuration resource which can be defined by different sources?
